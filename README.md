# ♟️ Smart Chess Move Predictor

An AI-powered chess move prediction system built using Flask and TensorFlow/Keras.  
The application allows users to play chess in the browser while the AI predicts optimal moves based on trained game data.

---

## 🚀 Features

- Interactive chess board interface
- AI-based move prediction
- Chess puzzle solving mode
- Game history storage
- Machine learning model trained on chess datasets

---

## 🛠 Technologies Used

- Python
- Flask
- TensorFlow / Keras
- HTML
- CSS
- JavaScript
- SQLite

## ⚙️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/Prathikarai/chess-move-predictor.git
```

### 2. Navigate to project folder

```
cd chess-move-predictor
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

### 4. Run the Flask server

```
python app.py
```

### 5. Open the application

```
http://127.0.0.1:5000
```
